## InputNumberWithUnit

基于 InputNumber 的二次封装，实现带单位的数字输入框

<Playground>
  <InputNumberWithUnit defaultValue={1000} unit="元" />
  <br />
  <InputNumberWithUnit defaultValue={1000} unit="$" />
</Playground>

## API

支持原 [InputNumber](https://ant-design.gitee.io/components/input-number-cn/) API

|参数|说明|类型|默认值|
|:--|:--|:--|:--|
|unit|单位|string|无|

> 注意：特定的货币符号会被前置，且金额在展示上会进行千位符格式化，不影响获取的值。

## 支持的货币符号

```
¥：人民币/日元
$：美元
NT$：新台币
zł：波兰兹罗提
€：欧元
```
