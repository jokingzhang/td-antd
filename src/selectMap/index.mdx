## SelectMap

基于 Select 的二次封装

<Playground>
  <SelectMap data={{ key1: 'Jack', key2: 'Lucy' }} />
  <SelectMap data={['Jack', 'Lucy']} />
  <SelectMap data={[{ name: 'Jack', age: 18 }, { name: 'Lucy', age: 20 }]} fields={['age', 'name']} />
</Playground>

## API

支持原 [Select](https://ant-design.gitee.io/components/select-cn/) API

|参数|说明|类型|默认值|
|:--|:--|:--|:--|
|data|渲染所需要的数据源|object / array|-|
|fields|当 data 为数组时，且数组成员为对象时，需要申明渲染字段|[string, string]|-|

> 注意：默认宽度是 100%
